Interface of the SWITCHcast Recorder

Recorder Interface

The recorder interface is made up of several blocks. Elements marked with an 2 exist twice, one each for the first and second audio and video input source.

Preview Window2
Two windows showing the currently recorded content of the left and right video input source
Video Source Selector2
Dropdown list showing all available video input sources (Default: None)
Audio Source Selector2
Dropdown list showing all available audio input sources (Default: None)
Audio Volume Control2
Slider for adjusting the volume of the recording, only visible if an audio source has been selected
Audio Level Indicator2
, only visible if an audio source has been selected
Mute Button2
